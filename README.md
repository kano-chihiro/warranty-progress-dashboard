# 保証・ハード進捗ダッシュボード

`db.noesiskai.com` で運用している本番ダッシュボードです。

画面は「保証」と「ハード」の二つです。

有料サービスモードは 2026年7月22日に廃止しました。

過去の有料サービスデータは D1 に保存したままとし、画面には表示しません。

## ハード部門の区分

- 新品カメラ
- 新品レンズ
- 中古カメラ・レンズ

中古カメラと中古レンズは予算と実績を分けず、一つの「中古ハード」として集計します。

## デプロイ

`main` ブランチへの push を起点に、GitHub Actions が Cloudflare Pages の `warranty-progress-dashboard` へデプロイします。

## 印刷契約

プレビューと印刷結果は、A4 横一枚に収めます。

レイアウトを変えたときは、保証とハードの両方を PDF 化し、ページ数と描画結果を確かめます。
